Classic Snake Game

Control the snake, eat food, and grow longer while avoiding collisions. How long can you survive?

How to Play

Use Arrow Keys (↑↓←→) to control the snake
Or use WASD keys (W=Up, A=Left, S=Down, D=Right)
On mobile: Use the directional buttons below the game
Press SPACEBAR or Pause button to pause/unpause
Eat red food to grow and increase your score (+10 points)
Game speed increases every 50 points
Avoid hitting walls or yourself - Game Over!
Try to beat your high score and set new records
